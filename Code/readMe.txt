How to install E-Dent web application:

1- Install EDent2020 folder on htdocs
2-Open xampp on your device
2- Start Apache and MySQL
3- Implement database file through import (doctors_appointment.sql)
4- Go to localhost/EDent2020/index.php

E-Dent system is an web application, you can find it online on
